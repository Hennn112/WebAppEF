# WebAppEF (.NET Core MVC + EF Core)

## Cara Jalanin
1. Clone repo
2. Jalankan perintah:
```bash
dotnet restore
dotnet ef database update
    Pastikan kamu sudah install EF CLI jika belum:
    dotnet tool install --global dotnet-ef
dotnet run
```
